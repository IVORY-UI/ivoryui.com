Develop your web pages faster and easier.

Highly extensible, modular and open-source CSS framework. Prebuilt with grid system and flexible components helps you to develop powerful interfaces.
Get Started

No JavaScript required

IVORY is a CSS only framework, so it can be integrated in any JS environment.

Ready to use

IVORY gives you a collection of beautiful, responsive & fully functional components ready to be put to use. Helps you to build your pages faster and easier.

Everything responsive

Don't worry about writing complex media queries in your CSS. IVORY focuses mobile first and helps you to build responsive pages easily.

Flexible & Extensible

IVORY is completely flexible to style your pages. It's easy to enhance or tweak any component as you need.
